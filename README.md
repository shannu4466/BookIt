# Welcome to BookIt project

## Project info

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone https://github.com/Shannu-BookIt/BookIt

# Step 2: Navigate to the project directory.
cd BookIt

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## BookIt – Your All-in-One Ticket Booking Platform

BookIt is a full-featured ticket booking platform built using React and TypeScript, designed to simplify and enhance the booking experience for Movies, Events, Plays, and Sports. Whether you’re a movie buff, an event-goer, or a sports fan, BookIt brings everything under one roof with a clean UI, secure authentication, and a robust admin panel.

## 🔥 Features

## 🧑‍💻 User Features

🔐 Authentication – Sign up, login, Google Sign-In, password reset, and profile management.

🎟️ Book Tickets – Effortlessly book tickets for movies, events, plays, and sports.

📍 Location-Based Filtering – View content and events available in your city.

📊 Trending & Offers Section – Discover popular shows and avail exclusive offers.

✏️ Editable Profiles – Update profile info with safe cancel/save controls that don’t overwrite data unless confirmed.

## 🛠️ Admin Features

📥 Content Management – Add or modify movies, plays, events, and sports from the admin dashboard.
